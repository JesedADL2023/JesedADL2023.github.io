Trabajos realizados

Desafío 1 - Mi Curriculum Vitae

Como primera actividad, se invita a realizar una pequeña actividad que consiste en armar tu CV en HTML 5. Utilizaremos la estructura básica de HTML y todas las etiquetas que hemos aprendido en esta sesión. Hay libertad de acción para agregar el contenido básico que se estime conveniente. Un desafío es una actividad integral que busca poner en práctica todo lo aprendido en la unidad y en el curso hasta el momento. Aunque sea básico, nos remitimos a lo reapsado en clases.

Desafío 2 - Construyendo un landing page

En este desafío validaremos nuestros conocimientos de CSS 3, trabajados en clases y en la guía. Para lograrlo, necesitaremos implementar CSS utilizando archivos hojas de estilo externas al documento HTML. Además, debemos estructurar una página en términos y assets del proyecto, además de implementar bordes, margen, padding y selectores de etiqueta o clase según sea necesario.

Desafío 3 - Iguana page

En este desafío validaremos nuestros conocimientos de posicionamiento en CSS. Para lograrlo, necesitaremos diferenciar los valores asociados a display (inline, block, inline-block), utilizar floats para posicionar texto e imágenes, y resolver el problema de colapso de floats utilizando un clearfix. Pondremos en este nuevo desafío a prueba nuestros conocimientos de la propiedad display y float, html semántico.

Desafío 4 - Cuppon

En el siguiente desafío validaremos nuestros conocimientos de Bootstrap 5. Para lograrlo, necesitaremos implementar el sistema de grillas de Bootstrap, distribuyendo el contenido adecuadamente, para hacer una página web adaptable a distintos dispositivos, aplicando las clases de Bootstrap a través de los componentes card y formularios, utilizando la documentación disponible. Se dispondrá de una maqueta matríz.

Desafío 5 - Github

En este desafío validaremos nuestros conocimientos de Github. Ahora necesitaremos aplicar las etapas del versionamiento de GIT, para mantener un repositorio de versiones, y el procedimiento de subida del código versionado mediante una conexión SSH, para la mantención de un repositorio remoto. Aplicaremos nuestros conocimientos sobre Git, Github, HTML y CSS. Se hará un procedimiento llamado fork.

Desafío Final - Desarrollo de portafolio

En este último desafío necesitaremos aplicar el paso a paso del trabajo con Github Pages y GIT Remoto, estableciendo un flujo de trabajo colaborativo a partir de un caso. Como último desafío de este módulo, debemos dejar nuestros trabajos subidos como cual portafolio a Github, y dejándolo disponible vía Github Pages. Además de generar colaboración en equipo.
